## Start with WorldWise App

1. Clone this project, cd to project

```
git clone https://github.com/BCIamLong/WorldWise.git

cd WorldWise
```

2. Run install for all packages

```
npm i

npm install
```

3. Start fake server:Æ°

```
npm run server
```

4. Start client

```
npm run dev
```
